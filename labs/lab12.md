# Laboratorium 12: Praca z formatem JSON

## Cel zajęć
Wykorzystanie formatu JSON do przesyłania i przechowywania danych.

## Zadania
1. Stwórz plik `dane.json`, w którym zapiszesz informacje o 3 książkach (tytuł, autor, rok wydania).
2. W języku JavaScript stwórz obiekt reprezentujący samochód. Użyj `JSON.stringify()`, aby zamienić go na tekst i wyświetlić w konsoli przeglądarki.
3. Napisz skrypt w Pythonie, który odczyta plik `dane.json` i wypisze tylko tytuły książek.
4. Użyj darmowego API (np. [JSONPlaceholder](https://jsonplaceholder.typicode.com/todos/1)), aby pobrać dane w formacie JSON (użyj modułu `requests` w Pythonie lub `fetch` w JavaScript).
5. Napisz program w Pythonie, który pobiera od użytkownika dane o nowej książce i dopisuje je do istniejącego pliku `dane.json`.
