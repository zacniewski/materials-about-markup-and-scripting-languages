# Języki znaczników i skryptowe

Materiały dydaktyczne dla przedmiotu "Języki znaczników i skryptowe" (30h wykładów, 30h ćwiczeń).

## Wstęp

### Czym są języki skryptowe?
Języki skryptowe to języki programowania zaprojektowane głównie do automatyzacji zadań, które w przeciwnym razie musiałyby być wykonywane ręcznie przez użytkownika lub inny program. Charakteryzują się one zazwyczaj:
- **Interpretacją:** Kod jest wykonywany bezpośrednio przez interpreter, bez konieczności wcześniejszej kompilacji do kodu maszynowego.
- **Wysokim poziomem abstrakcji:** Skupiają się na łatwości użycia i szybkości tworzenia kodu kosztem wydajności czysto obliczeniowej.
- **Dynamicznym typowaniem:** Często nie wymagają jawnego deklarowania typów zmiennych.

Przykłady: Python, JavaScript, Bash, Ruby.

### Czym są języki znaczników?
Języki znaczników (markup languages) służą do opisywania struktury i prezentacji danych wewnątrz dokumentu tekstowego. Wykorzystują one specjalne "znaczniki" (tagi), które instruują program odczytujący (np. przeglądarkę internetową), jak interpretować daną część tekstu. Języki znaczników nie są językami programowania w ścisłym tego słowa znaczeniu, ponieważ nie zawierają logiki sterującej (pętli, instrukcji warunkowych).

Przykłady: HTML, XML, Markdown.
